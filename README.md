This site was bootstrapped with [Lumen](https://github.com/alxshelepenok/gatsby-starter-lumen), a lightweight [Gatsby](https://www.gatsbyjs.com/) blog starter..

It is statically hosted on [Netlify](http://netlify.com/).

## Installation

1. Clone this repo
2. `npm install`
3. `gatsby develop`
